# Threading and recursive Locks

## About

This application demonstrates the usage of
[`RecursiveLock`](https://ariel-os.github.io/ariel-os/dev/docs/api/ariel_os/thread/sync/struct.Semaphore.html)s
as synchronization method for threads.

## How to run

In this directory, run

    laze build -b nrf52840dk run
