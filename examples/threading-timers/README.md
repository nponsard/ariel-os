# threading-timers

## About

This application demonstrates how to use timers with threads.
It starts a thread that prints a message ten times in 100ms intervals.

## How to run

In this directory, run

    laze build -b nrf52840dk run

## Expected output

The output of this example is:

    [INFO] Hello from main thread!
    [INFO] Hello again #0
    [INFO] Hello again #1
    [INFO] Hello again #2
    [INFO] Hello again #3
    [INFO] Hello again #4
    [INFO] Hello again #5
    [INFO] Hello again #6
    [INFO] Hello again #7
    [INFO] Hello again #8
    [INFO] Hello again #9
